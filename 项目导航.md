## 🎯 工作流程

### 标准工作流

```
1. 查看项目状态
   ↓
   PROJECT-STATUS.md

2. 执行当前任务
   ↓
   根据"当前任务"执行

3. 完成后更新状态
   ↓
   编辑 PROJECT-STATUS.md
   使用"更新状态模板"

4. 提交Git
   ↓
   git add .
   git commit -m "完成XXX"
   git push

5. 查看下一步
   ↓
   PROJECT-STATUS.md → "下一步要干嘛"
```

## 💡 使用建议

### 每天开始工作时
1. 打开 `PROJECT-STATUS.md`
2. 查看"当前任务"
3. 查看"下一步要干嘛"
4. 开始工作

### 完成任务后
1. 打开 `.github-template/更新状态模板.md`
2. 复制模板
3. 填写完成内容
4. 更新到 `PROJECT-STATUS.md`
5. 提交Git

### 不知道做什么时
1. 打开 `PROJECT-STATUS.md`
2. 查看"下一步要干嘛"
3. 按优先级执行
