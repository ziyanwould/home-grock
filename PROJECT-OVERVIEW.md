# 🎯 Home Grock 项目完全指南

Home Grock 是一个**个人财务管理和家庭物品库存系统**，帮助用户管理物品、避免浪费、理清债务、规划现金流。

## 🎯 核心功能

### 1. 物品管理
- **扫码识别**：商品条形码、药品追溯码、AI 拍照
- **快速记录**：一扫即录，自动填充物品信息
- **分类管理**：药品、食品、日用品等多分类
- **过期提醒**：提前 7/3/1 天提醒快过期的物品
- **AI 分析**：购物建议、库存分析、浪费预警

### 2. 财务管理
- **智能记账**：手动记账、AI 小票识别、账单导入
- **债务追踪**：支持 6 种还款方式
- **还款计划**：自动生成每期还款明细
- **提前还款**：计算能省多少利息
- **现金流预测**：12 个月前瞻预测
- **财务分析**：月度统计、分类分析、趋势图表

### 3. 资产管理
- **资产盘点**：年终大扫除，自动对比差异
- **标签生成**：批量二维码标签，便于查找
- **家庭协作**：邀请家人共同管理

## 💡 项目亮点

### 1. 完整的解决方案
不只是单一功能，而是完整的个人财务管理系统：
- 物品管理 + 财务管理 + 债务管理
- 三位一体，帮助用户全面理财

- 购物建议（根据库存推荐）

- 扫码快速记录（不需要手动输入）
- 自动生成还款计划（复杂计算自动化）

- 数据加密存储
- 不分享给第三方
- 不用于广告
- 可以随时导出和删除

### 5. 完全免费
- 没有付费功能
- 没有广告
- 未来会提供 Docker 镜像自建

## 🚀 技术栈

### 前端
- **框架**：UniApp + Vue 3
- **打包**：Webpack / Vite
- **UI**：原生微信小程序 UI + Web 自定义 UI

### 后端
- **运行时**：Node.js
- **框架**：Express
- **数据库**：MySQL 8.0 + Redis
- **ORM**：Sequelize
- **存储**：MinIO（对象存储）

### AI/ML
- **供应商**：硅基流动（国内可用）
- **模型**：DeepSeek-V3（文本）+ Qwen3-VL-235B（视觉）
- **功能**：商品识别、小票识别、智能分析

### 部署
- **服务器**：自建或云服务
- **容器**：Docker（后续计划开源镜像）
- **CDN**：可选（用于静态资源加速）
