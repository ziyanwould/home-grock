# 3分钟快速上手

## 第1分钟：注册登录

### 微信小程序（推荐）

1. 打开微信，扫描二维码（放在README）
2. 点击"微信登录"授权
3. 创建或加入家庭

**提示**：创建家庭后，可以邀请家人一起使用

### Web管理后台

1. 访问 https://web.home-grock.com
2. 使用微信扫码登录
3. 开始使用

---

## 第2分钟：记录第一个物品

### 方法1：扫码添加（推荐）⭐

1. 点击首页"扫码"按钮
2. 对准商品条码扫描
3. AI自动识别并填充信息
4. 确认后点击"保存"

**支持识别**：
- 商品条形码（食品、日用品等）
- 药品追溯码（药品包装上的20位数字）
- 二维码

### 方法2：拍照识别

1. 点击"添加物品"
2. 上传商品照片
3. AI自动识别商品信息
4. 确认后保存

### 方法3：手动添加

1. 点击"添加物品"
2. 手动输入物品信息
   - 物品名称（必填）
   - 分类（食品、日用品、药品等）
   - 数量和单位
   - 过期日期（重要！）
   - 存放位置
3. 点击"保存"

---

## 第3分钟：记录第一笔消费

### 方法1：AI小票识别（推荐）⭐

1. 点击"记账" → "AI识别"
2. 拍摄超市小票照片
3. AI自动识别商品清单
4. 确认后点击"导入"

**提示**：
- 小票要拍清楚，光线充足
- 支持大部分超市小票（沃尔玛、永辉、盒马等）

### 方法2：手动记账

1. 点击"记账" → "添加交易"
2. 选择来源账户（如：微信）
3. 选择目标账户（如：食品支出）
4. 输入金额和日期
5. 点击"保存"

---

## 第4分钟：添加债务信息

### 记录信用卡/贷款

1. 打开"财务管理" → "贷款管理"
2. 点击"添加贷款"
3. 填写信息：
   - **贷款名称**：如"招行信用卡分期"
   - **原始本金**：借了多少钱
   - **年化利率**：多少%（看贷款合同）
   - **还款方式**：
     - 等额本息（房贷常用）
     - 等本等息（信用卡分期常用）
     - 其他方式
   - **总期数**：分多少期还
   - **首次还款日**：第一次还款的日期
4. 点击"保存"

**系统会自动生成还款计划！**

---

## 🎉 恭喜！你已经学会了核心功能

### 接下来可以：

#### 设置过期提醒

1. 打开"设置" → "提醒管理"
2. 勾选"过期提醒"
3. 选择提醒时间（提前7天/3天/1天）
4. 开启微信订阅消息

**效果**：药品、食品快过期时自动提醒你

#### 查看数据分析

1. 打开"统计分析"
2. 查看：
   - 物品价值统计（家里的东西值多少钱？）
   - 过期物品统计（浪费了多少？）
   - 消费分析（钱都花在哪了？）
   - 债务分析（总负债、每月还款）

#### 邀请家人

1. 打开"家庭管理"
2. 点击"邀请成员"
3. 分享邀请码给家人
4. 家人扫码加入

**好处**：
- 家人可以协同管理物品
- 共享物品信息（避免重复购买）
- 共同查看家庭财务状况

## 💡 使用技巧

### 技巧1：给物品拍照存档

添加物品时，建议拍照保存：
- 记录物品外观
- 快速识别（不用看字）
- 方便闲置转卖

### 技巧2：设置合理的分类

建议分类：
- 食品（调料、零食、饮料）
- 药品（感冒药、创可贴、维生素）
- 日用品（洗发水、纸巾、洗衣液）
- 化妆品（护肤品、彩妆）
- 其他

**分类越细，统计越准确**

### 技巧3：记录购买价格

添加物品时，记录购买价格：
- 统计家庭资产
- 分析消费习惯
- 闲置转卖参考

### 技巧4：定期盘点

建议每季度或半年盘点一次：
- 检查哪些物品快过期
- 整理闲置物品
- 更新库存数量

## ❓ 遇到问题？

### 扫码识别不出来？

**可能原因**：
- 条码太小或模糊 → 换个角度重新扫
- 光线不足 → 到光线好的地方
- 不支持的条码类型 → 使用拍照识别或手动添加

### AI识别不准确？

**可能原因**：
- 照片模糊 → 重新拍清楚的照片
- 光线太暗 → 到光线好的地方拍照
- 商品比较特殊 → 手动修改识别结果

### 忘记家庭邀请码？

打开"家庭管理" → "邀请码" → 查看或重新生成
