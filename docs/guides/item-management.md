# 物品管理完全指南

> 如何避免药品、调料批量过期，避免重复购买浪费钱

## 为什么要管理物品？

很多人（包括我）的问题：
- **药品囤太多**，结果批量过期
- **调料开了忘记用**，半年后发霉
- **不知道家里有什么**，重复购买浪费钱
- **东西放哪忘记了**，找不到又去买

**Home Grock 帮你解决这些问题。**

---

## 第一步：把物品录入系统

### 方法1：扫码识别（最快）⭐

**适合**：有条形码的商品（食品、日用品、药品）

**操作步骤**：
1. 打开首页，点击"扫码"
2. 对准商品条形码扫描
3. AI自动识别并填充信息：
   - 商品名称
   - 品牌
   - 规格
   - 参考价格
   - 分类
4. 确认信息无误，点击"保存"

**支持的条码类型**：
- ✅ 商品条形码（13位数字，如6901234567890）
- ✅ 药品追溯码（20位数字）
- ✅ 二维码

**识别速度**：< 3秒

[GIF动画演示：扫码识别]

---

### 方法2：拍照识别

**适合**：没有条码或条码不清楚的商品

**操作步骤**：
1. 点击"添加物品"
2. 点击"拍照"或"从相册选择"
3. AI识别商品信息（需要3-5秒）
4. 确认信息，点击"保存"

**AI能识别什么**：
- 商品名称
- 品牌
- 规格
- 生产日期
- 保质期

**提示**：
- 拍清楚商品包装正面
- 光线充足
- 包含商品名称和日期信息

[截图：拍照识别示例]

---

### 方法3：小票批量导入⭐

**适合**：刚从超市买回来的商品

**操作步骤**：
1. 打开"记账" → "AI识别"
2. 拍摄超市小票
3. AI自动识别所有商品：
   - 商品名称
   - 数量
   - 价格
4. 勾选想要添加的物品
5. 点击"导入到物品库"

**优势**：
- 一次导入多个商品
- 自动记录价格
- 关联消费记录

**支持的超市**：
- 沃尔玛、永辉、盒马、大润发
- 大部分连锁超市小票

[截图：小票识别]

---

### 方法4：手动添加

**适合**：扫码和拍照都无法识别的商品

**操作步骤**：
1. 点击"添加物品"
2. 手动填写信息：
   - **物品名称**（必填）
   - **分类**（食品、药品、日用品等）
   - **数量和单位**（如：2瓶）
   - **购买日期**
   - **生产日期**
   - **过期日期**（重要！）
   - **购买价格**
   - **存放位置**（如：厨房、冰箱、药箱）
   - **备注**
3. 点击"保存"

## 第二步：设置过期提醒

### 为什么要设置提醒？

**避免批量过期浪费**：
- 药品过期 → 扔掉几百块
- 调料过期 → 心痛
- 食品过期 → 浪费

**提前提醒 → 及时用掉 → 不浪费**

### 如何设置提醒？

#### 全局提醒设置

1. 打开"设置" → "提醒管理"
2. 开启"过期提醒"
3. 选择提醒时间：
   - 提前30天
   - 提前15天
   - 提前7天（推荐）⭐
   - 提前3天（推荐）⭐
   - 提前1天
   - 当天过期
4. 开启微信订阅消息（重要！）

#### 单个物品提醒设置

1. 打开物品详情
2. 点击"提醒设置"
3. 设置该物品的专属提醒时间
4. 保存

**适用场景**：
- 重要药品（提前30天提醒）
- 贵重食品（提前7天提醒）
- 普通物品（提前3天提醒）

## 第三步：日常使用技巧

### 技巧1：买东西前先查库存

**场景**：去超市购物，想买酱油

**操作**：
1. 打开首页"搜索"
2. 输入"酱油"
3. 查看家里还有几瓶

**结果**：
- ✅ 家里有 → 不买了，省钱
- ❌ 家里没有 → 买

**每次这样做，能避免重复购买浪费**

---

### 技巧2：快过期的优先用

**场景**：收到提醒"蚝油还有3天过期"

**操作**：
1. 今晚做菜时优先用这瓶蚝油
2. 用完后在系统里标记"已消耗"
3. 更新库存数量

**结果**：不浪费，省钱

---

### 技巧3：定期盘点

**建议频率**：
- 药品：每季度盘点一次
- 食品/调料：每月盘点一次
- 日用品：每季度盘点一次

**如何盘点**：

#### 方法A：完整盘点

1. 打开"盘点管理"
2. 点击"创建盘点"
3. 选择范围（全部、按房间、按分类）
4. 开始盘点
5. 逐个扫码确认
6. 完成盘点

#### 方法B：快速盘点（推荐日常使用）

1. 打开首页
2. 点击"快速确认"
3. 扫码物品
4. 系统自动更新"最后确认时间"

**好处**：
- 定期整理，心里有数
- 发现过期及时处理
- 发现闲置可以转卖

---

### 技巧4：分类管理

**建议分类**：

**食品类**：
- 调料（酱油、醋、蚝油、豆瓣酱）
- 零食（饼干、薯片、巧克力）
- 饮料（可乐、果汁、牛奶）
- 粮食（大米、面粉、食用油）
- 速食（方便面、罐头）

**药品类**：
- 感冒药
- 肠胃药
- 止痛药
- 外用药（创可贴、碘伏）
- 保健品（维生素、钙片）

**日用品类**：
- 洗护（洗发水、沐浴露、牙膏）
- 清洁（洗衣液、洗洁精、纸巾）
- 护肤（面霜、面膜、防晒）
- 其他

**分类越细，统计越准确，查找越方便**

---

### 技巧5：记录存放位置

**为什么要记录位置**：
- 找东西快
- 避免"明明有，就是找不到"

**如何记录**：
添加物品时，在"存放位置"填写：
- 厨房 - 冰箱
- 厨房 - 调料架
- 卧室 - 床头柜
- 卫生间 - 镜柜
- 客厅 - 茶几
- 药箱 - 一层/二层

**查找时**：
搜索物品 → 看存放位置 → 直接去拿

## 第四步：消耗和补充

### 消耗物品

**场景**：用完了一瓶洗发水

**操作**：
1. 打开物品详情
2. 点击"消耗"
3. 选择消耗数量
4. 保存

**系统会**：
- 自动减少库存数量
- 如果数量为0，标记为"已用完"
- 记录到使用日志

---

### 补充物品

**场景**：又买了2瓶酱油

**操作**：
1. 搜索"酱油"
2. 打开物品详情
3. 点击"补充"
4. 输入新增数量：2
5. 输入购买日期和过期日期
6. 保存

**系统会**：
- 自动增加库存数量
- 更新最新过期日期
- 记录购买历史

## 第五步：数据分析

### 查看物品统计

**打开"统计分析" → "物品统计"**

你会看到：
- **物品总数**：家里一共有多少东西
- **物品总价值**：家里的东西值多少钱
- **即将过期**：有多少东西快过期了
- **已过期**：有多少东西已经过期（要处理）

### 查看过期统计

**打开"统计分析" → "过期统计"**

你会看到：
- 近30天过期了哪些东西
- 浪费了多少钱
- 哪个分类浪费最多

**反思**：
- 为什么会过期？
- 下次少买一点？
- 提前提醒时间改长一点？

## 真实案例

### 案例1：药品管理

**用户**：小王，28岁

**之前的问题**：
- 家里药箱有20多种药
- 经常买重复的
- 等要用的时候找不到
- 大扫除发现一堆过期药

**使用后**：
1. 把所有药品扫码录入
2. 设置提前30天提醒
3. 买药前先查库存
4. 快过期的药提前用掉

**结果**：
- 半年没扔过过期药了
- 买药前查一下，没买重复的
- **一年省了500多块**

---

### 案例2：厨房调料管理

**用户**：小李，32岁，爱做饭

- 调料瓶太多，不知道有什么

2. 设置提前7天提醒
3. 做菜前看看快过期的

**结果**：
- 半年没扔过过期调料了
- 知道家里有什么，不乱买

## 常见问题

### Q: 扫码识别不准确怎么办？

**A**：AI识别后可以手动修改：
1. 修改物品名称
2. 修改分类
3. 修改过期日期
4. 保存

### Q: 没有条码的物品怎么办？

**A**：使用拍照识别或手动添加

### Q: 如何批量导入物品？

**A**：使用小票识别功能，一次导入多个

### Q: 过期提醒在哪里看？

**A**：
- 首页会显示即将过期的物品
- 微信会推送提醒消息
- "统计分析"可以查看所有快过期的

### Q: 如何删除物品？

**A**：
1. 打开物品详情
2. 点击右上角"..."
3. 选择"删除"
4. 确认

**提示**：建议不要删除，而是标记为"已消耗"，这样可以保留历史记录
