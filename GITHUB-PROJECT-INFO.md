**Q: README 和 README.en.md 应该选哪个？**
A: 都保留。GitHub 会默认显示 README.md，用户可通过底部链接切换。

**Q: Topics 应该怎么选？**
A: 选择最能代表项目的标签。我们的 10 个标签涵盖：
- 功能（财务、物品、债务、支出）
- 技术（Vue3、Node.js）
- 属性（中文、开源、小程序、Web）

**Q: 是否需要翻译所有文档？**
A: 不需要。目前保持中文为主，英文 README 作为国际用户入口。

**Q: GitHub Pages 需要配置吗？**
A: 可选。如果想部署静态网站可以配置，但项目本身不需要。
