# 📝 项目状态更新模板

> 每次完成任务后，复制这个模板到 PROJECT-STATUS.md 的"更新记录"部分

---

## 更新模板（复制使用）

```markdown
### YYYY-MM-DD（日期）

**完成**：
- ✅ 任务1
- ✅ 任务2
- ✅ 任务3

**当前任务**：
- ⏳ 正在做的事情

**遇到的问题**：
- ⚠️ 问题描述（如果有）

**下一步**：
- 📌 下一个要做的任务
- 📌 另一个任务

**数据**（可选）：
- 截图完成：X/25
- GitHub Stars：X
- 用户注册：X
```

## Git提交模板

```bash

git add screenshots/

# 更新文档
git add docs/
git commit -m "Update user guides"
git push

# 添加案例

git commit -m "Add user case: 省钱案例"

# 更新状态

```
