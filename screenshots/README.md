# 产品截图说明

> 需要拍摄的产品截图清单

## 📸 拍摄要求

### 基本要求
- 高清截图（至少 1080p）
- 界面干净（没有敏感信息）
- 数据真实（不要空数据）
- 光线充足（看得清楚）

### 数据准备
建议提前录入一些测试数据：
- 10-15个物品（不同分类）
- 3-5笔贷款
- 10-20笔消费记录
- 有几个快过期的物品

## 制作工具

### 截图
- **macOS**: Command + Shift + 4
- **Windows**: Win + Shift + S
- **手机**: 电源键 + 音量键

### GIF录制
- **macOS**: Gifski (推荐)
- **Windows**: ScreenToGif
- **在线**: ezgif.com

### 图片编辑
- 压缩：TinyPNG
- 裁剪：预览（macOS）/ 画图（Windows）
- 标注：Xnip (macOS) / Snipaste (Windows)
